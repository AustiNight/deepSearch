# Changelog

## 0.0.0 - 2026-02-18
Version tag: v0.0.0

### Added
- Always-available **New Search** reset control that cancels in-flight runs and clears run state without clearing saved settings or API keys.
- Universal settings sync across devices via Cloudflare Access identity; **Save Configuration** syncs non-secret settings to the cloud and reports success/conflicts/errors.
- Transparency Map rendered as a full-width, normalized table with sticky headers and bullet-list cells for multi-value fields.
